# TCP와 UDP

데이터를 전달하기 위해 사용하는 프로토콜로, 전송 계층에 해당.

TCP(Transmission Control Protocol) : 연결 지향형 프로토콜

UDP(User Datagram Protocol) : 데이터를 데이터그램단위로 전송하는 프로토콜
